About RunCal

Tech Stack

  • Backend: Flask
  • Machine Learning: scikit-learn
  • Frontend: HTML, CSS, Bootstrap
  • Data Storage: Pickle files for model and scaler

What It Does

RunCal is a web application designed to predict the number of calories burned based on user inputs. It takes four key parameters: weight, height, duration of exercise, and speed. The application leverages a machine learning model to provide personalized calorie burn estimates.

How It Works

Users input their weight, height, duration of exercise, and speed into the form on the main page. Upon submission, the input data is processed and transformed using a scaler that standardizes the data. The transformed data is then fed into a pre-trained machine learning model to predict the number of calories burned. The prediction is then displayed to the user.

Model Details

The application uses a machine learning model trained to predict calorie expenditure. The model was trained using historical exercise data and uses features such as weight, height, duration, and speed. The model was serialized into a pickle file and is loaded at runtime to make predictions.

The scaler used for preprocessing the input data was also trained and saved. This ensures that the input data is transformed in the same way as the data used during model training, providing accurate predictions.
